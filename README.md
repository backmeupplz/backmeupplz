### 🤖 [SuperHarness](https://github.com/backmeupplz/superharness)
**Autonomous multi-agent orchestrator for AI coding agents via tmux**

### 🥁 [Drumkit](https://github.com/backmeupplz/drumkit)
**Low-latency TUI MIDI drum sampler for electronic drum kits**

Terminal-based MIDI drum sampler with minimal latency, perfect for electronic drum kits. Features sample drum kits available at [drumkit-kits](https://github.com/backmeupplz/drumkit-kits).

### 🎵 [DrumTutor](https://github.com/backmeupplz/drumtutor)
**Interactive web-based drum tutor**

Upload any MIDI file, listen, practice with real-time feedback, and master drums at your own pace. A browser-based drum learning platform.
